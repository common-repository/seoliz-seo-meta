== Description ==

Description ve keywords meta etiketlerini eklenti sayesinde site uzerinde dinamik olarak kullanabileceksiniz. Her yazi ve sayfa icin ayri ayri degerlerde aciklamalar ve anahtar kelimeler girebileceksiniz. Tum bu islemleri 3 KB boyutunda kucuk bir php dosyasi yapmaktadir. Sunucunuzu ve veritabaninizi yormayacak kod dokumu ile eklentimiz indirilebilir durumda.

== Installation ==

Eklentimizin kurulusuda normal diger eklentilerin kurulusu gibidir.

1. 'seoliz-seometa' klasorunu '/wp-content/plugins/' dizinine yukleyin
2. WordPress 'Eklentiler' panelinden eklentiyi aktiflestirin
3. Kurulum basariyla tamamlandi, kullanmanin tadini cikartin.
